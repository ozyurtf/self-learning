Truck Backer-Upper

A self-learning neural network agent that backs a truck-trailer to a loading dock using curriculum learning — no labeled data, no supervision.

How it works

Two neural networks collaborate: an Emulator that learns to predict the next truck state given a steering action, and a Controller that learns which action to take at each step. The controller is trained by backpropagating through the frozen emulator — no environment interaction is needed during controller training.

Both models are trained progressively through 10 curriculum lessons, starting with the truck nearly aligned and close to the target, and gradually expanding to arbitrary starting positions and angles. By lesson 10, the agent navigates successfully even from positions it has never seen during training.

Emulator

Forward model predicting the next cab and trailer state (x, y, θ₀, θ₁) from the current state and steering angle φ.

Controller

Policy network that outputs the steering angle at each step. Trained end-to-end through the emulator without any labeled demonstrations.

Curriculum Learning

10 progressively harder lessons — expanding x range, lateral offset, cab angle, and cab-trailer angle difference across lessons.
